1️⃣ Grundlagen der Java-Programmierung

Die wichtigsten Konzepte zum Start

📝 Was ist Java?

Java ist eine objektorientierte, plattformunabhängige Programmiersprache.

  • Entwickelt 1995 von Sun Microsystems
  • Motto: "Write Once, Run Anywhere" (WORA)
  • Läuft auf der Java Virtual Machine (JVM)
  • Jetzt von Oracle entwickelt

📦 Struktur eines Java-Programms

public class HelloWorld { public static void main(String[] args) { System.out.println("Hello from Java!"); } }
Wichtig:
  • class - Definition einer Klasse
  • main() - Einstiegspunkt des Programms
  • public - Zugriffsmodifikator (öffentlich)
  • static - Gehört der Klasse, nicht der Instanz

🔄 Compilation und Ausführung

// Kompilieren javac HelloWorld.java // Ausführen java HelloWorld
Schritt Beschreibung Ergebnis
Quellcode .java Datei HelloWorld.java
Kompilieren javac Compiler HelloWorld.class
Ausführen JVM interpretiert Ausgabe
💡 Tipp: Der Dateiname MUSS mit dem Klassennamen übereinstimmen! Wenn die Klasse "HelloWorld" heißt, muss die Datei "HelloWorld.java" sein.

4️⃣ Kontrollstrukturen

Verzweigungen und Schleifen

❓ if-else Anweisung

// if-else mit Alternative if (alter >= 18) { System.out.println("Du darfst es!"); } else { System.out.println("Zu jung!"); } // if-else if-else (mehrfache Auswahl) if (noten == 1) { System.out.println("Sehr gut!"); } else if (noten == 2) { System.out.println("Gut!"); } else { System.out.println("Nicht bestanden"); }

🔀 switch Anweisung

switch (wochentag) { case 1: System.out.println("Montag"); break; case 2: System.out.println("Dienstag"); break; default: System.out.println("Unbekannt"); break; }
⚠️ Wichtig: break; nicht vergessen! Sonst läuft der Code weiter (Fall-Through).

🔁 for-Schleife

// Klassische for-Schleife for (int i = 0; i < 5; i++) { System.out.println("i = " + i); // Gibt 0,1,2,3,4 aus } // Array durchlaufen int[] zahlen = {10, 20, 30}; for (int i = 0; i < zahlen.length; i++) { System.out.println(zahlen[i]); }
💡 Tipp: zahlen.length gibt die Länge des Arrays zurück!

↩️ while und do-while

while
Bedingung wird VOR dem Durchlauf geprüft
int x = 0; while (x < 3) { System.out.println(x); x++; }
do-while
Bedingung wird NACH dem Durchlauf geprüft
int x = 0; do { System.out.println(x); x++; } while (x < 3);

5️⃣ Arrays (Felder)

Sammlungen von Variablen

📋 1D-Arrays (eindimensional)

// Deklaration und Initialisierung int[] zahlen = {1, 2, 3, 4, 5}; // Deklaration mit Größe int[] noten = new int[6]; // Mit Werten füllen noten[0] = 1; noten[1] = 2; // Zugriff auf Elemente System.out.println(zahlen[0]); // Gibt 1 aus System.out.println(zahlen.length); // Gibt 5 aus
⚠️ Index: Arrays beginnen bei 0! Ein Array mit 5 Elementen hat Indizes 0-4.

📊 2D-Arrays (zweidimensional)

// Matrix 2x3 (2 Zeilen, 3 Spalten) int[][] matrix = { {1, 2, 3}, {4, 5, 6} }; // Zugriff System.out.println(matrix[0][1]); // Gibt 2 aus System.out.println(matrix[1][2]); // Gibt 6 aus // Mit Schleifen durchlaufen for (int i = 0; i < matrix.length; i++) { for (int j = 0; j < matrix[i].length; j++) { System.out.print(matrix[i][j] + " "); } }

🔍 Array-Operationen

// Maximum finden double max = 0; for (int i = 0; i < noten.length; i++) { if (noten[i] > max) { max = noten[i]; } } // Array ausgeben System.out.println(java.util.Arrays.toString(zahlen)); // Array sortieren (wenn import vorhanden) java.util.Arrays.sort(zahlen);

🎯 Enhanced for-Schleife (for-each)

int[] zahlen = {1, 2, 3, 4, 5}; // Einfacher als normales for - kein Index nötig for (int zahl : zahlen) { System.out.println(zahl); }
💡 Tipp: Verwende for-each, wenn du keinen Index brauchst!

6️⃣ Objektorientierung (OOP)

Klassen, Attribute, Konstruktoren und Vererbung

🏗️ Klassendefinition mit Attributen

Beispiel: Adresse-Klasse

public class Adresse { // Private Attribute private String strStrasse; private String strHausNr; private String strPLZ; private String strOrt; // Getter public String getStrStrasse() { return strStrasse; } // Setter public void setStrStrasse(String strStrasse) { this.strStrasse = strStrasse; } public String getStrHausNr() { return strHausNr; } public void setStrHausNr(String strHausNr) { this.strHausNr = strHausNr; } public String getStrPLZ() { return strPLZ; } public void setStrPLZ(String strPLZ) { this.strPLZ = strPLZ; } public String getStrOrt() { return strOrt; } public void setStrOrt(String strOrt) { this.strOrt = strOrt; } }
⚠️ Wichtig: this.strStrasse = strStrasse; - this verweist auf das Attribut der Klasse, der Parameter ist die Übergabe!

🔨 Konstruktoren und super()

Basis-Klasse (Person):

public class Person { private static int iPAnzahl = 0; // Static: gehört der Klasse private int iPNr; private String strNachName; private String strVorName; // Komposition: Objekt von anderer Klasse Adresse adresse = new Adresse(); // Konstruktor public Person() { iPNr = ++iPAnzahl; // Jede neue Person erhält eine Nummer } // Methode zum Eingeben public void erfasseDaten() { Scanner sc = new Scanner(System.in); System.out.println("Dateneingabe:"); System.out.print("Nachname:\t"); strNachName = sc.next(); System.out.print("Vorname:\t"); strVorName = sc.next(); System.out.print("Strasse:\t"); adresse.setStrStrasse(sc.next()); System.out.print("Hausnr:\t\t"); adresse.setStrHausNr(sc.next()); System.out.print("PLZ:\t\t"); adresse.setStrPLZ(sc.next()); System.out.print("Ort:\t\t"); adresse.setStrOrt(sc.next()); } // Überladene Methode (gleicher Name, andere Parameter) public void erfasseDaten(String strNachName) { this.strNachName = strNachName; } // Methode zum Ausgeben public void druckeAusweis() { System.out.println("Personennr.:\t" + iPNr); System.out.println("Nachname:\t" + strNachName); System.out.println("Vorname:\t" + strVorName); System.out.println("Adresse:\t" + adresse.getStrStrasse() + " " + adresse.getStrHausNr() + "\n\t\t" + adresse.getStrPLZ() + " " + adresse.getStrOrt()); } }

🔄 Vererbung (extends) und super

Schüler-Klasse erbt von Person:

import java.util.Scanner; public class Schueler extends Person { // Erbt von Person private String strKlasse; // Konstruktor public Schueler() { // super() wird automatisch aufgerufen } // Überschriebene Methode @Override public void erfasseDaten() { Scanner sc = new Scanner(System.in); super.erfasseDaten(); // Ruft erfasseDaten() der Person auf System.out.print("Klasse:\t\t"); strKlasse = sc.next(); System.out.println(); } // Überschriebene Methode @Override public void druckeAusweis() { System.out.println("Schülerausweis"); super.druckeAusweis(); // Ruft druckeAusweis() der Person auf System.out.println("Klasse:\t\t" + strKlasse); System.out.println(); } }

Lehrer-Klasse erbt ebenfalls von Person:

import java.util.Scanner; public class Lehrer extends Person { // Erbt von Person private String strFach; // Konstruktor public Lehrer() { // super() wird automatisch aufgerufen } // Überschriebene Methode @Override public void erfasseDaten() { Scanner sc = new Scanner(System.in); super.erfasseDaten(); // Ruft erfasseDaten() der Person auf System.out.print("Fach:\t\t"); strFach = sc.next(); System.out.println(); } // Überschriebene Methode @Override public void druckeAusweis() { System.out.println("Lehrerausweis"); super.druckeAusweis(); // Ruft druckeAusweis() der Person auf System.out.println("Fach:\t\t" + strFach); System.out.println(); } }
💡 super: Ruft die Methode der Basisklasse auf! Wichtig für Vererbung!

✅ Praktisches Beispiel: SchoolApp

import java.util.Scanner; public class SchoolApp { public static void main(String args[]) { // Objekte erzeugen Schueler oSchueler1 = new Schueler(); Schueler oSchueler2 = new Schueler(); Lehrer oLehrer1 = new Lehrer(); // Daten eingeben oSchueler1.erfasseDaten(); oSchueler2.erfasseDaten(); oLehrer1.erfasseDaten(); // Ausweise drucken System.out.println("\n--- AUSWEISE ---\n"); oSchueler1.druckeAusweis(); oSchueler2.druckeAusweis(); oLehrer1.druckeAusweis(); } }

🔐 Zugriffsmodifikatoren

Modifikator Sichtbarkeit Verwendung
public Überall Methoden, die von außen aufrufbar sein sollen
private Nur in der Klasse Attribute, interne Methoden (Encapsulation)
protected Klasse + Unterklassen Für Vererbung (extends)
- Paket Default (keine Angabe)

📌 Komposition vs. Vererbung

Komposition (HAS-A):
Eine Klasse ENTHÄLT ein Objekt einer anderen Klasse
public class Person { Adresse adresse = new Adresse(); // Person HAT eine Adresse }
Vererbung (IS-A):
Eine Klasse IST eine spezielle Art einer anderen Klasse
public class Schueler extends Person { // Schüler IST eine Person }

7️⃣ Methoden, return und Benutzereingaben

Funktionen mit Rückgabewerten und Scanner

📝 Methodendefinition mit return

Syntax:

[Zugriffsmodifikator] [static] Rückgabetyp Methodenname(Parameter) { // Methodenrumpf return [Wert]; // Rückgabewert }

Beispiele mit unterschiedlichen Rückgabetypen:

// Rückgabewert: int public int addieren(int a, int b) { return a + b; // Gibt Ergebnis zurück } // Rückgabewert: double public double berechneFlaeche(double radius) { return 3.14159 * radius * radius; } // Rückgabewert: String public String getName() { return strName; // Gibt den Namen zurück } // Rückgabewert: boolean public boolean istAelter(int alter) { return alter >= 18; // true oder false }

❌ vs ✅ void vs. return

void - Keine Rückgabe
public void drucken() { System.out.println( "Hallo!"); // Kein return! } // Aufruf drucken(); // Keine Zuweisung möglich!
mit return - Mit Rückgabe
public int berechnen() { int ergebnis = 5 + 3; return ergebnis; } // Aufruf int x = berechnen(); System.out.println(x); // 8
⚠️ Wichtig: void-Methoden können nicht einer Variablen zugewiesen werden! int x = drucken(); führt zu Fehler!

📥 Scanner für Benutzereingaben

Scanner importieren und verwenden:

import java.util.Scanner; public class Eingabe { public static void main(String[] args) { // Scanner erzeugen Scanner sc = new Scanner(System.in); // Ganzzahl einlesen System.out.print("Alter: "); int alter = sc.nextInt(); // Text einlesen (bis zur Leerzeichen) System.out.print("Name: "); String name = sc.next(); // Ganze Zeile einlesen (mit Leerzeichen) System.out.print("Adresse: "); String adresse = sc.nextLine(); // Dezimalzahl einlesen System.out.print("Preis: "); double preis = sc.nextDouble(); // Ausgabe System.out.println("Name: " + name + ", Alter: " + alter); } }
💡 Scanner-Methoden:
  • sc.nextInt() - Ganze Zahl
  • sc.nextDouble() - Dezimalzahl
  • sc.next() - Einzelnes Wort (bis Leerzeichen)
  • sc.nextLine() - Ganze Zeile

🔄 Praktisches Beispiel: Daten erfassen und ausgeben

import java.util.Scanner; public class Schueler extends Person { private String strKlasse; @Override public void erfasseDaten() { Scanner sc = new Scanner(System.in); // Basisklasse Daten erfassen super.erfasseDaten(); // Zusätzliche Daten System.out.print("Klasse:\t\t"); strKlasse = sc.next(); System.out.println(); } @Override public void druckeAusweis() { System.out.println("Schülerausweis"); super.druckeAusweis(); System.out.println("Klasse:\t\t" + strKlasse); System.out.println(); } // Getter mit return public String getKlasse() { return strKlasse; } }

📌 static vs. Instanz-Methoden

Art Gehört Beispiel Aufruf
static Der Klasse public static void main() Klasse.methode()
Instanz Dem Objekt public void fahren() objekt.methode()
public class Auto { private static int iPAnzahl = 0; private String farbe; // Static Methode - gehört der Klasse public static int getAnzahl() { return iPAnzahl; } // Instanz-Methode - gehört jedem Auto-Objekt public void fahren() { System.out.println(farbe + " Auto fährt!"); } } // Verwendung int anzahl = Auto.getAnzahl(); // Static Auto a = new Auto(); a.fahren(); // Instanz

⚡ Überladung (Overloading)

Gleicher Methodenname, unterschiedliche Parameter:

public class Person { private String strNachName; // Methode ohne Parameter public void erfasseDaten() { Scanner sc = new Scanner(System.in); System.out.print("Name: "); strNachName = sc.next(); } // Überladene Methode mit Parameter public void erfasseDaten(String strNachName) { this.strNachName = strNachName; } } // Aufruf Person p = new Person(); p.erfasseDaten(); // Interaktive Eingabe p.erfasseDaten("Mueller"); // Direkte Übergabe
💡 Java wählt automatisch die richtige Methode basierend auf den Parametern!

8️⃣ Prüfungstipps & Häufige Fehler

Was ihr beachten müsst!

✅ Das solltet ihr können

  • ✔️ Einfache Programme schreiben (mit main-Methode)
  • ✔️ Variablen deklarieren und initialisieren
  • ✔️ Unterschied zwischen primitiven und Referenztypen
  • ✔️ if-else und switch Anweisungen verwenden
  • ✔️ for, while und do-while Schleifen nutzen
  • ✔️ Arrays erstellen und durchlaufen
  • ✔️ Klassen mit Attributen und Methoden definieren
  • ✔️ Objekte mit new erzeugen
  • ✔️ Methoden mit Parametern und Rückgabewert schreiben

❌ Häufige Fehler

  • ❌ Array Index - Vergessen, dass Arrays bei 0 beginnen!
  • ❌ Division durch Null - 5 / 0 führt zu ArithmeticException
  • ❌ break in switch - Ohne break läuft der Code weiter!
  • ❌ Semikolon - Jede Anweisung muss mit ; enden
  • ❌ Klammern - { } und ( ) müssen paarweise vorkommen
  • ❌ == vs .equals() - Für Strings .equals() verwenden!
  • ❌ NullPointerException - Auf null prüfen vor Zugriff
  • ❌ return in void - void-Methoden können keine Werte zurückgeben!
  • ❌ Scanner ohne import - Immer import java.util.Scanner; schreiben!
  • ❌ super.methode() vergessen - In Unterklassen die Basisklasse aufrufen!

⚡ Code-Qualität

// ❌ FALSCH - Schlechte Variablennamen int x = 25; int y = 100; // ✅ RICHTIG - Aussagekräftige Namen int alter = 25; int gehalt = 100; // ❌ FALSCH - Keine Kommentare for (int i = 0; i < 5; i++) { System.out.println(i); } // ✅ RICHTIG - Mit Kommentar // Gebe Zahlen von 0-4 aus for (int i = 0; i < 5; i++) { System.out.println(i); }

📚 String-Operationen (wichtig!)

String text = "Hallo"; // Länge System.out.println(text.length()); // 5 // Verbinden (Konkatenation) String name = "Max"; String gruß = "Hallo " + name; // "Hallo Max" // Vergleichen String a = "Hallo"; String b = "Hallo"; System.out.println(a.equals(b)); // true System.out.println(a == b); // Kann false sein! // Zu Großbuchstaben System.out.println(text.toUpperCase()); // "HALLO" // Zu Kleinbuchstaben System.out.println(text.toLowerCase()); // "hallo" // Teilstring System.out.println(text.substring(1, 4)); // "all"
